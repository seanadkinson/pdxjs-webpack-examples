Loaders Galore

Webpack is very well written and modular. The chain of loaders need only end with valid javascript, and literally anything can be required!

less-loader

Compile LESS CSS at bundle-time

    require('../css/styles.less');

coffee-loader

Easily compile and include your coffeescript files

    var cafe = require('./file.coffee');

jade-loader

Compile your Jade templates, if you are into that kind of thing

    var template = require('./template.jade');

jsx-loader

Automatically parse your JSX syntax in our React components! Also, no need to add the /** @jsx React.DOM */ comment at the top of the files

    var React = require('react');
    var UserPanel = require('./UserPanel.jsx');
    React.renderComponent('body', <UserPanel/>);

url-loader

Returns a URL to a resource, which can be used for links or images

    var imageSrc = require('../img/logo.png');
    $('img.logo').attr('src', imageSrc);
