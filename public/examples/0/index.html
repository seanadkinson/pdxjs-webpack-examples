CommonJS Refresher

CommonJS modules can:

  • Use require() to import other modules 
        var dependency = require('../lib/myDependency.js');
  • Set exported properties on the special exports object 
        exports.myPublicProperty = "some value";
  • Set the entire exports object on module.exports 
        module.exports = function() { /* do something */ }

Simple Example

colors.js

    exports.sky = 'blue';
    exports.grass = 'green';

main.js

    var colors = require('./colors.js');
    console.log('The sky is: ' + colors.blue);
